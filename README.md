# dokun-ui

# Install dependencies
```bash
sudo apt install build-essential cmake git libx11-dev libgl1-mesa-dev libglu1-mesa-dev libvulkan-dev libglfw3-dev libopenal-dev
```

# Build dokun-ui
```bash
mkdir build && cd build
cmake ..
make
```
